⚙️ COMO FUNCIONA
──────────────────────────────────────────────────────────────────────────────

1️⃣  Você faz push para GitHub
    ↓
2️⃣  GitHub Actions dispara automaticamente
    ↓
3️⃣  npm install → instala dependências
    ↓
4️⃣  npm run generate-data → CSV vira JavaScript
    ↓
5️⃣  npm run build → Vite otimiza tudo
    ↓
6️⃣  Deploy para branch gh-pages
    ↓
7️⃣  GitHub Pages publica em https://seu-usuario.github.io/seu-repo/


🚀 3 MANEIRAS DE COMEÇAR
──────────────────────────────────────────────────────────────────────────────

✅ OPÇÃO 1 - RÁPIDA (1 minuto) - RECOMENDADO
   chmod +x setup.sh
   ./setup.sh
   # Preencha username e repo name
   # Pronto!

✅ OPÇÃO 2 - MANUAL (5 minutos)
   1. Edite package.json (linha homepage)
   2. Edite vite.config.js (linha base)
   3. git init && git add . && git commit -m "init"
   4. git remote add origin https://github.com/seu-usuario/seu-repo.git
   5. git push -u origin main

✅ OPÇÃO 3 - DETALHADA (15 minutos)
   Leia arquivo GITHUB_PAGES_SETUP.md
